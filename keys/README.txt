Place your Apple private key file (AuthKey_2G3MFFZW92.p8) in this directory.

The private key file should have been downloaded from App Store Connect when you created your API key.
It contains content that looks like:

-----BEGIN PRIVATE KEY-----
MIGTAgEAMBMG...
[many lines of base64 encoded content]
...
-----END PRIVATE KEY-----

This file should be kept secure and should not be committed to version control.
